# QR-reader
A python script which allows you to read any kind of QR code and even barcodes!

## Features
       
2. Easy to use
   
3. Also powerful
   
4. Not PNG ? No problem

Want to have it ? Let's go for installation, don't worry it will be pretty easy

## Installing

Simply clone the repo:

```shell
git clone https://github.com/TheOddZer0/QR-Reader
```
Then install with pip:
```shell
pip install .
```

## Usage

simply run `qr-reader`, followed by the picture name, also you can call that script from any place

For example `qr-reader test.png` (this command will not fail because there is a test.png file included)

The `test.png` was generated by [QR-Creator](https://github.com/TheOddZer0/QR-Creator)
